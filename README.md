# csh Starter Overlay

This overlay prepares the base project starter for the ContextVM SSH project.

Use it with `templates/project-starter/`:

1. Copy the base starter into the new project.
2. Overlay the files from `templates/csh-starter/`.
3. Start the project with a seed prompt and project-specific references.

This overlay is intentionally small. It specializes startup guidance, communication logging, and
research focus without forking the whole base starter.

## Included Files

- `AGENTS.md`
- `agent-brief`
- `handoff.md`
- `docs/process/project-profile.md`

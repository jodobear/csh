# Phase 0: Bootstrap Prompt

Goal: establish the project profile, research layout, first build-plan baseline, and initial
handoff.

## Inputs

- `AGENTS.md`
- `docs/process/process-principles.md`
- `docs/process/project-profile.md`
- `docs/process/research-workflow.md`
- `docs/plans/decision-log.md`
- `docs/plans/build-plan.md`
- `handoff.md`

## Required Work

- fill in the project profile
- confirm canonical references and quality gates
- prepare `docs/references/local/` for project-created research
- create the first research note if research is the immediate next step
- update the build-plan baseline and handoff state

## Required Output

- updated `docs/process/project-profile.md`
- updated `docs/plans/build-plan.md`
- updated `handoff.md`
- optional first note under `docs/references/local/`

## Clarifying Question Gate

- stop if high-impact ambiguity remains
- ask one targeted question before advancing

## Exit Criteria

- project profile is explicit
- research workflow is ready to use
- first baseline plan exists
- handoff reflects the current state
